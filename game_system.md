# AI育成ゲーム - 転生編 システム仕様書 (Master Specification)

## 1. プロジェクト概要
クォータービュー（アイソメトリック）のマップ上で、独自のステータスと欲求を持つAIキャラクターを育成・観察するシミュレーションゲーム。
HTML5 Canvas と Vanilla JavaScript を用いて構築されており、MVCアーキテクチャに近いモジュール分割がなされている。
「転生編」として、ゲーム開始時の性格診断による初期キャラクター決定や、全11種族・多段階分岐に完全対応した壮大な進化システムを搭載。
さらに、フィールドでのランダムな出会いから始まる**「師匠・弟子入りシステム（全6職業のクエスト進行）」**や、過去に育成したキャラクターたちを鑑賞・放置できるエンドコンテンツ「放牧システム」が組み込まれている。

## 2. ファイル構成
* `test_game.html` / `style.css`: UIの土台、各種メニュー（カジノ、図鑑、ステータス詳細、進化ルーレット、**エンカウント会話・クエスト報告UI**など）の定義、天候エフェクトのアニメーション定義。
* `main.js`: ゲームループ、初期化、モード切替。AI調整用の直接入力UIパネル、フレームのコピー＆ペースト、WASDキー長押しでの快適な座標微調整など、強力な開発・デバッグ機能を含む。
* `system.js`: グローバル変数、マップの自動生成ロジック、セーブ/ロード機能、進化先リストの動的生成、性格診断ロジックの管理、多言語対応。
* `data.js`: 画像ソース、マップチップ、施設・アイテムのマスターデータ。全11種族の進化ツリーや、8種類の性格に応じた専用セリフ・アクション画像マッピング定義を含む。
* `ai_core.js`: 【コアロジック】AIのステータス管理、高精度な経路探索、タスクキューの消化。**師匠クエストの進行管理（ランク1〜9の課題定義と達成判定）、エンカウント（出会い）発生ロジック、作業時の素材自動補充システム（師匠の支援）**を含む。
* `game_manager.js`: 建築予定地の探索、農業アクション、アイテム売買、借金・自己破産処理。
* `view_renderer.js`: Canvasへのクォータービュー描画、カメラ追従、キャラクターのアニメーション、各種作業のプログレスゲージ可視化。
* `ui_controller.js`: HTML/DOM操作、各種パネルのUI更新。豪華な進化ルーレット演出、**チャット入力による言葉の学習と行動指示の判定**、ステータス画面でのクエスト進捗・ライセンス表示制御を担う。
* `feature_pokedex.js`: 発見したキャラを閲覧する図鑑機能。放牧マップへの配置・入場用UIの動的生成を含む。
* `feature_grazing.js`: 【エンドコンテンツ】放牧システムの完全独立型ディレクターロジック。キャラクターの自律作業アニメーション制御、オフライン放置ボーナスの計算、アイテム一括回収機能。

## 3. コアデータ構造
ゲーム内の全ての状態は、主に以下のグローバル変数で管理される。

### 3.1 `assets` (マップオブジェクト群)
* マップ上に存在する全てのオブジェクト（地形、水、橋、施設、自然、**師匠のキャンプ地など**）を格納する連想配列。
* プロパティ: `type`, `dx/dy`, `sx/sy/sw/sh`, `scale`, `durability` など。

### 3.2 `aiPet` (AIキャラクター状態)
* AIキャラクターの全ステータスと行動状態を保持するオブジェクト。
* **欲求・ステータス**: `energy`, `hunger`, `stats` (intel, power, mood, beauty), `darknessCounter`。
* **進行管理**: `age`, `generation`, `gold`, `inventory`, `skills` (建築, 料理, 鍛冶の熟練度)。
* **システム管理**: `baseType`, `currentSkin`, `discoveredMonsters`, `learnedWords` (学習した言葉リスト)。
* **弟子入り状態 (`apprentice`)**: `currentMaster` (現在の師匠), `rank` (各職業のランク), `activeQuest` (受注中の課題), `qVal` (課題の進捗度), `isExcommunicated` (破門フラグ), `isGraduated` (免許皆伝フラグ) などを保持。

### 3.3 `grazingData` (放牧マップ状態)
* 放牧用マップ（最大5つ）のデータを保持。本編とは完全に独立した箱庭空間を構成する。

## 4. AIのタスク管理・思考ルーチン詳細 (`ai_core.js`)

### 4.1 状態遷移 (`actionState`)
* `idle` (待機), `moving` (移動中), `entering`/`inside` 等の空間遷移、および各種作業ステート (`farming_work`, `cooking`, `smithing`, `building` 等) を持つ。

### 4.2 チャット指示と言葉の学習
* チャットで入力されたテキストから特定のキーワード（「料理」「鍛冶」「建築」など）を抽出し、AIがその言葉を学習する。
* 学習済みの言葉で指示を出された場合のみ、それを「タスク（仕事）」として `schedule` 配列に追加し、実行に向かう。

### 4.3 自由行動と性格システム (Idle時の振る舞い)
* `schedule` が空の時、能力値の伸び方に応じて**8種類の性格（average, gloom, scholar, athlete, idol, artist, stoic, lazy）**が動的に判定される。
* 判定された性格に応じ、**学習済みの言葉の範囲内**で、一定確率で自発的にタスク（勉強、筋トレ、鍛冶、探検など）をスケジュールに組み込んで実行する。

## 5. 各種ゲームシステム詳細

### 5.1 農業・建築・探索・生産システム
* **農業**: 畑への種まき、水分管理、害虫退治、収穫。
* **建築**: チャット指示により、手持ちの素材（木材・石）を消費してAIの近くの空き地に自動で施設を建築する。
* **探索**: 施設への侵入、ステータス依存の成功判定、耐久値（durability）の減少と跡地化。
* **生産（料理・鍛冶・釣り）**: 熟練度に応じた成功率判定。作業中は専用のアニメーションと進捗ゲージが表示される。

### 5.2 師匠・弟子入りシステム（職業クエスト）
* **エンカウントと弟子入り**: フィールドを歩いていると、一定確率で6種の職業（冒険家、農家、漁師、料理人、鍛冶師、建築士）のマスターと遭遇する。ステータスと学習単語に基づく「入門試験」に合格すると弟子入りできる。
* **報告先の固定出現**: 弟子入りすると、マップ上の空き地に「師匠のテント」や「移動レストラン」などの拠点が永続的に出現し、いつでも報告に行けるようになる。
* **ランクと課題（クエスト）**: 各職業にはランク1〜9の課題（特定ステータスへの到達、作業の回数こなし、特定アイテムの作成・納品など）が用意されており、チャットで「〇〇のところへ」と指示して報告を行う。
* **素材の自動補充（救済措置）**: 弟子入り中、対象の作業（料理、鍛冶、釣り、建築など）を行おうとした際に素材や道具が不足している場合、詰み防止のために師匠が裏で自動的に必要最低限の素材（ニンジン、鉄鉱石、古い釣り竿など）を支給してくれる。
* **免許皆伝と破門**: ランク9の最終試練をクリアすると「免許皆伝」となり、大幅なステータスボーナスと専用の称号を獲得する。課題を途中で投げ出すと破門されるが、基礎トレーニングを繰り返すことで復帰可能。

### 5.3 経済・天候・進化システム
* **経済**: ショップでの売買、救助物資（借金）、時間経過による自己破産（強制リセット）。
* **天候**: 時間経過に伴う昼夜・季節・天候変化と、Canvas上のオーバーレイエフェクト。
* **進化**: 全11系統・多段階分岐の進化ツリー。条件を満たすと豪華なルーレットUIとクロスフェード演出を経て姿が変化する。

### 5.4 【エンドコンテンツ】放牧・放置システム (`feature_grazing.js`)
* **マップ解放と配置**: 図鑑の発見数に応じて最大5つの放牧マップが解放される。育成済みのキャラクターをマップにつき3体まで配置可能。
* **ステータス引継ぎ**: 配置されたキャラは、本編でセーブされた「その姿での最終ステータス・熟練度」を引き継いで活動する。
* **鑑賞とアイテム回収**: 放牧されたAIたちは独自のディレクターロジックにより自律的に活動し、集めたアイテムはUIから「一括回収」して本編へ持ち込み可能。放置ボーナス（オフライン報酬）も完備。

## 6. デバッグ・エディタ機能（AI調整機能）
* **AI Adjust Mode**: キャラクターのアニメーション座標（`sx`, `sy`, `sw`, `sh`）をリアルタイム調整。
* ドロップダウンによるアクション一発指定、Shift+Tabでの逆順移動、WASDキー長押し対応による直感的な微調整。
* `Ctrl+C / Ctrl+V` によるフレーム座標のコピー＆ペースト。`localStorage` と `data.js` の安全なマージ処理（アクション用追加画像の保護）。
